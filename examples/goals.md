# Atris Goals

## Current North Star

Build the task-native AI operating system for real companies.

## Active Goals

- Make tasks the atomic unit of human and agent work across CLI, desktop, cloud, Swarlo, and RL evaluation.
- Keep every task small enough to understand in one screen, but rich enough to carry context, proof, review, and the next task.
- Make local work durable first: SQLite state, append-only events, projection JSON, regenerated TODO, and review episodes.
- Make the Command Center show who is doing what, what needs review, what is blocked, and what should start next.
- Connect local task events to cloud company truth through safe sync planning before enabling live mutation.
- Use task review episodes as the evaluation substrate for recursive self-improvement.
- Prefer proof-backed work over claims: every completed task should leave a command, artifact, receipt, customer signal, or human review.

## Tonight's Loop

```text
pick next task
  -> claim it
  -> do the smallest useful slice
  -> attach proof
  -> review with reward + lesson
  -> create or suggest the next sharper task
  -> refresh projection
```
